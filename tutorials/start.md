## Getting Started

### Installing Ark JS

Using [npm](https://nodejs.org/en/download/):

```sh
$ npm install arkjs
```

Using [yarn](https://yarnpkg.com/en/docs/install):

```sh
$ yarn add arkjs
```

### Using Ark JS

#### Client

```html
<script src="node_modules/arkjs/app.js"></script>
```

#### Server

##### JavaScript

```js
const ark = require('arkjs')
```

##### ECMA 2015+ and TypeScript

```js
import * as ark from 'arkjs'
```
